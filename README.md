## Overview

A complete, structured **Python Full Course** repository covering **31 lessons** across 4 phases — from data types and loops to OOP, DSA, decorators, file handling, regular expressions, and SQLite.

This repository is designed for learners who want a clean, well-organized reference with runnable code examples, commented explanations, real-world mini projects, and a structured study roadmap.

---

## What's Covered

### Phase 1 — Python Foundations
- Variables, data types, collections (list, tuple, set, dict)
- Conditionals, loops, break, continue
- Pattern programs and number programs
- Functions — def, return, \*args, \*\*kwargs
- Lambda, map(), filter()
- Problem solving miscellaneous

### Phase 2 — Core Concepts
- Recursion — base case, call stack
- String programs and hollow patterns
- List programs and matrix operations
- OOP — classes, objects, static methods
- OOP — encapsulation and inheritance
- Decorators, polymorphism and abstraction

### Phase 3 — Advanced Python
- File handling — read, write, append, `with` statement
- JSON — `json.dumps()`, `json.dump()`, `json.load()`
- Regular expressions — `re` module, match, search, findall, sub
- Copy operations — shallow copy vs deep copy
- Exception handling — try, except, else, finally
- Iterators and generators — `__iter__`, `__next__`, `yield`
- SQLite — `sqlite3`, CRUD operations
- OOP Bank Projects (Basic + Advanced)

### Phase 4 — DSA & Practice
- Searching — linear search, binary search, interpolation search
- Sorting — bubble, selection, insertion, quick sort
- DSA test cases — 40+ programs for interview practice
- Decorator & Singleton pattern — movie and ticket booking
- Module imports, number conversions, string & list snippets

## Getting Started

### Prerequisites

- Python 3.x
- Git

### Run Locally

1. **Clone the repository**
```bash
git clone https://github.com/sundarrajus/PYTHON.git
cd PYTHON
```

2. **Navigate to the course folder**
```bash
cd "Python Full Course"
```

3. **Run any lesson**
```bash
python 01_datatypes_and_collections.py
```

4. **Reference the PDF guide**

Open [`Python_Complete_Guide.pdf`](https://github.com/sundarrajus/PYTHON/blob/main/Python_Complete_Guide.pdf) for full concept notes.
